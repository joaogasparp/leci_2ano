# Second Year of Computer and Informatics Engineering (LECI)

Welcome to the repository for the second year of Computer and Informatics Engineering (LECI). This repository contains a collection of notes, projects, and assignments from various courses taken during the second year of the program.

## Overview

The second year of LECI continues to build on the foundational topics introduced in the first year, with a focus on more advanced concepts in computer science and engineering. The main subjects include:

### 1st Semester

- **Algoritmos e Estruturas de Dados** 
- **Competências Transferíveis I** 
- **Arquitetura de Computadores I** 
- **Mecânica e Campo Eletromagnético** 
- **Redes de Comunicações I** 

### 2nd Semester

- **Competências Transferíveis II** 
- **Análise de Sistemas** 
- **Arquitetura de Computadores II** 
- **Redes de Comunicações II** 
- **Sinais e Sistemas Eletrónicos** 

## How to Use

Feel free to browse through the materials, use them as study aids, or as references for your own coursework. Contributions and improvements to the content are welcome; please submit a pull request or open an issue if you have any suggestions.
